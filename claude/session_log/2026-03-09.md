## Design Decisions

- Used `subprocess.run` for pytest instead of programmatic pytest API — keeps
  the test runner isolated from the kernel's own process.
- `ChangeResult` is a dataclass, not a dict — gives downstream code type safety.
- Protected files checked by both resolved path and relative string — prevents
  symlink or path traversal bypasses.
