## Design Decisions

- run.py seeds kernel capabilities on first run so the gap detector doesn't
  immediately try to re-build existing kernel modules. Idempotent — subsequent
  runs skip already-registered components.
- Gate pre-flight checks use conservative cost estimates ($0.01 for plan,
  $0.02 for codegen). These are pessimistic enough to trigger budget gates
  before wasting a real API call.
- `get_task_config()` requires BOTH provider and model to be set for a task.
  If only one is set, it falls back to the default config entirely. This
  prevents half-configured routing from producing confusing errors.
